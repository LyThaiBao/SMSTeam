1. api về auth

	- POST auth/login : đăng nhập hệ thống 
          + các field bắt buộc: email, passWord.
	  + Trả về thông tin người dùng: { access_token , user : { email , name, role (STUDENT ADMIN LECTURER)  } }
	  + gửi về cookie tên "refreshToken" chứa refresh_token.
	
	- POST auth/refreshToken : lấy access_token mới khi token cũ đã hết hạng

2. faculty (khoa học)

	- POST faculty/ : tạo khoa mới
	  + các field bắt buộc: name, code (mã khoa) , image (file ảnh)
	  + dữ liệu trả về : { Success, data: { id , name , status , img (url) , code } }

	- GET faculty/ : lấy tất cả các khoa có trong hệ thống
	  + dữ liệu trả về: { quantity(số lượng khoa) , listFaculty : { id , name , status , img (url) , code }   }

	- GET faculty/:id : lấy khoa theo id
	  + field id gửi qua params
	  + Dữ liệu trả về: { data: {id , name , status , img (url) , code   }}

	- PATCH faculty/:id : sửa dữ liệu của khoa
 	  + field id gửi qua params
	  + các field có thể có: { image , name , code , status }
	  + trả về trạng thái cập nhật (thành công hoặc thất bại)

	- DELETE faculty : coming soon

3. major (ngành học)

	- POST major/ : tạo ngành học mới
	  + các field bắt buộc: name, faculty_id, required_credits, code
	  + dữ liệu trả về : { Success, data: { id , name , required_credits , code , faculty: { id , code , name } } }

	- GET major/ : lấy tất cả các ngành học có trong hệ thống
	  + dữ liệu trả về: { quantity(số lượng ngành) , listMajor : { id , name , required_credits, code, faculty: { name } }  }

	- GET major/:id : lấy ngành học theo id
	  + field id gửi qua params
	  + Dữ liệu trả về: { data: { id , name , required_credits, code, faculty: { name }}}

	- PATCH major/:id : sửa dữ liệu của ngành học
	  + các field có thể có: { name , code }
	  + trả về trạng thái cập nhật (thành công hoặc thất bại)
	  + field id gửi qua params


4. student_class (lớp chuyên ngành)

	- POST studentClass/ : Tạo lớp chuyên ngành mới
	  + field bắt buộc : name , major_id, code (mã lớp)
	  + dữ liệu trả về : { name , code ,  major: { code, name }  }
	
	- GET studentClass/ : Lấy tất cả các lớp chuyên ngành đang có
	  + dữ liệu trả về: { quantity , listStudentClass : { id , code , name , status , major : { mame , code } } }

	- GET studentClass/:id : lấy lớp chuyên ngành theo id
	  + id gửi qua params
	  + dữ liệu trả về : { id , name , major: { name , code } }

	- PATCH  studentClass/:id : cập nhật thông tin lớp chuyên ngành
	  + gửi id qua params
	  + các field có thể có: name , code , status
	  + dữ liệu trả về : dữ liệu mới ( sau cập nhật ) 

	- DELETE studentClass/:id : xóa lớp chuyên ngành ( khi không còn sinh viên )
	  + id gửi qua params 
	  + trả về trạng thái xóa (thành công hoặc thất bại)

5. student 
		
	- POST student/ : tạo sinh viên mới
	  + các field bắt buộc: email, passWord, role, userName, mssv, class_id, gender
	  + dữ liệu trả về : dữ liệu của sinh viên vừa tạo

	- GET student/ : lấy tất cả sinh viên đang có trong hệ thống
	  + dữ liệu trả về: { quantity, listStudent: { id , enable , userName , email , mssv , role , class_id(lớp chuyên ngành) , gender(Nam hoặc Nữ) , enroll_year (năm nhập học) , status }  }

	- GET student/:id : lấy sinh viên theo id
	  + dữ liệu trả về: { id , enable , userName , email , mssv , role , class_id(lớp chuyên ngành) , gender(Nam hoặc Nữ) , enroll_year (năm nhập học) , status }

	- DELETE student/:id : xóa sinh viên
	  + id gửi qua params

	- UPDATE student/:id 
	  + các trường có thể có: username, address , phone.
6. Lecturer

	- POST lecturer/ : tạo sinh viên mới
	  + các field bắt buộc: email, passWord, userName, faculty_id, gender
	  + dữ liệu có thể có: phone
	  + dữ liệu trả về : dữ liệu của giảng viên vừa tạo

	- GET lecturer/ : lấy tất cả giảng viên đang có trong hệ thống
	  + dữ liệu trả về: { quantity, listLecturer: { id, name, email, faculty_id: { name , code } , gender , phone } }

	- GET lecturer/:id : lấy giảng viên theo id
	  + dữ liệu trả về: { id, name, email, faculty_id: { name , code } , gender , phone }

	- DELETE lecturer/:id : xóa giảng viên
	  + id gửi qua params
	  + gửi về trạng thái xóa

	- UPDATE lecturer/:id  
	  + id gửi qua params
	  + các field có thể có : phone, userName
